# e-commerce-store




 ## Technology Which I Used
   ***Backend***
   
 - Nodejs
 - Mongodb
 - Express

 ***Frontend***
 

 - React Js
 - Redux

<br />

 ## Step to Start Project
 

 
**For Backend**
 1. `git clone https://github.com/MohitSojitra/e-commerce-store.git`
 2. `cd e-comerce-backend`
 3. `npm`
 4. `npm data:import`
 5. `npm dev`

**For Frontend**

 1. `cd e-comerce-frontend`
 2. `npm`
 3. `npm start`
