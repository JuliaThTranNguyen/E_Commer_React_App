# e-commerce-store




 ## Technology Which I Used
   ***Backend***
   
 - Nodejs
 - Mongodb
 - Express

 ***Frontend***
 

 - React Js
 - Redux

<br />

 ## Step to Start Project
 

 
**For Backend**
 `cd e-comerce-backend`
 `npm`
 `npm data:import`
 `npm dev`

**For Frontend**

`cd e-comerce-frontend`
`npm`
`npm start`
