# e-commerce-store




 ## Technology Which I Used
   ***Backend***
   
 - Nodejs
 - Mongodb
 - Express

 ***Frontend***
 

 - React Js
 - Redux

<br />

 ## Step to Start Project
 

 
**For Backend**
 1.`cd e-comerce-backend`
 2.`npm`
 3.`npm data:import`
 4.`npm dev`

**For Frontend**

5.`cd e-comerce-frontend`
6.`npm`
7.`npm start`
